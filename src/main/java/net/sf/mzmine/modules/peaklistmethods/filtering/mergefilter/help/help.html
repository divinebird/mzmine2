Merge filter

Description

Replaces similar peak-list rows with a single peak-list row. Similar peak-list rows are determined by comparing their peaks' RT and m/z values. If they differ by less than the amounts specified by the RT tolerance and m/z tolerance parameters then they can be merged.

Method parameters

Name suffix
This is the suffix used to identify the new peak list in Peak list frame of the application desktop.
m/z tolerance
Maximum m/z difference between peaks to be merged.
RT tolerance
Maximum retention time difference between peaks to be merged.
Require same identification
If the checkbox is selected then peaks must have the same identification for them to be merged.
Remove source peak list after merging
It the checkbox is selected the source peak list will be remove and the filtered version remains.
